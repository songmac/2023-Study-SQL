0. 데이터베이스
- 데이터 : 컴퓨터 안에 기록되어 있는 숫자
- 데이터베이스 : 데이터의 집합
- 저장 장소 : 하드디스크, 플래시메모리(SSD)

1. 시스템 내의 데이터베이스
- 데이터베이스는 개인용 컴퓨터나 휴대용 기기, 데이터센터 등 다양한 시스템에서 사용

2. DB와 DBMS
- DB : Database
- DBMS : Database Management System
- DBMS 사용 목적 : 생산성 향상, 기능성, 신뢰성 확보

3. 데이터베이스를 조작하는 언어 SQL
- 관계형 데이터베이스 관리 시스템(RDBMS:Relational Database Management System)조작에 사용
- IBM이 개발, ISO에 의해 표준화가 진행됨

3-1. SQL명령의 종류
*각각 독립적임
- DML(Data Manipulation Language) : 새롭게 데이터 추가, 삭제, 내용 갱신 등에 사용
- DDL(Data Definition Language) : 데이터베이스 객체라는 데이터 그릇을 이용하여 데이터를 만들거나 삭제
- DCL(Data Control Language) : 트랜잭션을 제어하는 명령과 데이터 접근권한을 제어하는 명령 포함
