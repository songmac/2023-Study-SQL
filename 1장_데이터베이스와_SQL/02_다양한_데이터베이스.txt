1. 데이터베이스 종류
- 데이터 저장 방법에 따라 몇가지로 분류
- 아래는 개발된 순서로 나열

1-1. 계층형 데이터베이스
- 폴더와 파일 등의 계층 구조로 데이터를 저장하는 방식
ex. 하드디스크, DVD 파일 시스템

1-2. 관계형 데이터베이스
- 표 형식의 데이터를 저장하고 각 표에 이름을 붙여 관리

1-3. 객체지향 데이터베이스
- 객체지향 언어(자바나 C++)를 이용하여 가능한 객체 그대로를 데이터베이스의 데이터로 저장하는 것

1-4. XML 데이터베이스
- 태그를 이용해 마크업 문서를 작성할 수 있게 정의 ex. HTML과 흡사
- SQL명령어 대신 XQuery라는 전용 명령어 사용

1-5. 키-밸류 스토어(KVS)
